- `[[ ref name ]]`為  Obsidian 應用軟體的特殊標記，用來表示該文件的超鏈接。
- 此為主卡的格式說明。
- 主卡的文件格式為 markdown（.md）。
- 主卡分為二個部分檔案名稱和檔案內容。檔案名稱為客製的時間戳記編號和實作標題組成。檔案內容的版面只有一個區塊，元資料區塊（放在 # metadata 之下，避免被其他的 header 語法判斷成其子內容）。
- 元資料區塊由兩個屬性組成，一個為"由此引用”，為鏈接到文獻卡，一個為"到此實作“，為鏈接到代碼卡。
- 文件名稱的格式為 YYYY_MMDD_HHmm_ss_main - 實作標題。其中實作標題為簡要說明實作的內容。範例：2022_0412_1852_00_main - 實作 Java Stream API 的各種做法
